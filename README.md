# Text Frequency
Small tool to count text frequency.

## Usage
```
$ frequency <words>
```

## Building
Requires [`dotnet`](https://dotnet.microsoft.com/download) to build.
```
$ git clone https://github.com/PotatoMaster101/txtfreq
$ cd txtfreq
$ dotnet build -c Release
```

## Testing
```
$ dotnet test
```
